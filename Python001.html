Python 笔记（一）

Date Wed 11 September 2013 By laoyi Category Python. Tags Python

本系列博文记录学习Python过程的笔记和想法，简单而狗血，杂乱而无章，主要目的是方便来日查看。

使用的Python版本为2.7.3， 在GNU/Linux Ubuntu 12.04LTS操作系统下实现和测试

脚本声明和中文编码

在Python脚本的首行和次行添加以下命令可在命令行调用，并设置编码为中文gb2312：

#!/usr/bin/env python
# -*- coding:gb2312 -*-

Python的注释以#开始，单行注释

单句Python语句占一行时结尾不用任何符号，多句时用分号分割，但极不推荐此种风格。

字符串

字符串有三种写法，单引号或双引号包围的单行字符串（可用反斜杠连接多行），或者三个单引号或双引号跨多行：

#简单的输出语句，包含三种字符串形式
print 'Hello,python!'
print u"中文是可以支持的."
print u'''字符串的第三种格式
跨行字符串:用三个单引号或双引号约束。
前方的u使该字符串作为unicode编码，正确处理中文。
'''

原始字符串

#普通字符串和原始字符串（r），原始字符串不进行自动转义
print "This is a normal string: C:\now\print\the\location\in\windows"
print r"This is a raw string with same content:C:\now\print\the\location\in\windows"

变量

Python的变量不用声明类型，据说这种风格为鸭子类型，关于鸭子类型以后研究清楚了再详细记录。算术操作与C++类似，简单的赋值等操作如下：

#单个菊花为乘法，双菊花为求幂
x = 5
print "%d ** 3 = " % x, x ** 3

y = "scutlaoyi.github.com"
print "%s is my blog~" % y

关于print的补充：print可类似C的printf，参数列表前用百分号，参数列表使用元组包括。单个参数可省略元组的括号。print也可连接多个元素，使用逗号分隔，其输出为同一行。

Python会在print后自动添加换行，方便操作，若不想换行，在print行最末添加逗号即可

print "这行不会自动换行",

命令行获取用户输入

获取用户输入有两种方式，input和raw_input，均读取一行， 但input将内容当作Python语句处理，raw_input直接当成字符串

#raw_input:获取用户输入，全部当成字符串
usrInput = raw_input("write something:")
print "you have typed:", usrInput

#input:获取用户输入，将内容当作Python语句
usrInNum = input("Give me a number:")
print "%d * 2 = " % usrInNum, usrInNum * 2

列表

Python拥有强大的内置数据结构，列表、元组和字典，类似于C++标准库中的vector，提供了丰富的功能。 列表与元组类似，但列表可更改，元组不可更改。列表用[]包括，元组用()包括。

关于列表的操作：

strList = list("hello")
print strList, "is from list('hello')"
aList = [1,2,3,4,5,6,7,8,9]
print "The list is :", aList
#正向索引
print "The second one in list:", aList[1]
#逆向索引，倒数第一个
print "The last one in list:", aList[-1]
#切片操作，从首个到第三个(不包含第四个)
print "The first to the third elements in list:", aList[0:3]
#步进为2
print "The first, third, fifth elements in list:", aList[0:6:2]
#
print "List length:%d" % len(aList)
print "Max number in list:", max(aList)
print "Min number in list:%d" % min(aList)
#列表的编辑操作
aList[1] = 10
print "The second one of list has been changed to %d" % aList[1]
del aList[5]
print "After deleting the sixth element, length of list is:", len(aList)
aList[5:] = [11, 12, 13, 14, 15]
print "Change from the sixth elements:", aList
aList.append(151)
aList.insert(3, 31)
bList = [10, 9, 8]
aList.extend(bList)
print "After appending and extending, the aList is:", aList
print "pop the last one:", aList.pop()
print "pop the first one:", aList.pop(0)
aList.remove(151)
print "remove the 151 in aList:", aList

#查找
ind = aList.index(10)
print "the index of 10 in aList is:%d" % ind
aList.sort()
print "sorted:", aList
aList.reverse()
print "reversed:", aList

元组

创建元组的几种方法：

tupleA = (1,)
tupleB = (1,2,3)
tupleC = tuple("Hello")

元组的基本操作与list类似，但不能更改，在此不再赘述。

字符串的补充方法

find：

strA = "Hey!My name is szy!"
ind = strA.find("My")
#ind == 4
ind = strA.find("scutLaoYi")
#ind == -1
ind = strA.find("My", 10, 14)
#ind == -1, 定义起点终点

join:

#注意join的另一方为字符串列表
seq = ['1', '2', '3', '4', '5']
result = '/'.join(seq)
#result == "1/2/3/4"

split:

#类似join的反向操作
print result.split('/')
#['1', '2', '3', '4']

lower:

upStr = "AbCdE"
print upStr.lower()
#abcde

upper:

print upStr.upper()
#ABCDE

replace:

print upStr.replace('A', 'a')
#abCdE

strip:

longStr = "  !! This is a string with some rubbish  ~~ "
print longStr.strip()
#!! This is a string with some rubbish.  ~~
print longStr.strip(' !~')
#This is a string with some rubbish.

其他大量内置的方法可查看documentation.
