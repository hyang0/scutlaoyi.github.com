  1. Bash笔记(一)

    四 02 一月 2014
    By 老逸

    本系列博文记录学习Linux终端Bash过程中的收获，主要为常用命令一览表及相关功能的记录，方便日后再次查看。

    基础知识

    以echo为例，bash的基本命令及结构如下：

    显示命令后的所有参数

    echo [option] [arg0] [arg1] [arg2] [...]

    命令、参数之间用空格区分，空格数量差异没有作用, 在单引号或双引号之间的所有字符被当成同一个参数

    echo "Hey, this is a single argument."

    设定option使echo解析字符操作符

    echo -e "A line with n a newline."

    命令执行成功返回0，否则返回非0值, 返回值的意义可查命令的man

    命令的帮助文档

    type

    命令有两种类型，内部和外部命令 显示命令的类型

    type [command]

    某些命令有两种类型，显示所有类型

    type -a [command]

    当执行同名命令时内置命令优先， 执行外部命令必须指定命令路径，如echo ...
