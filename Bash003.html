Bash笔记(三)

六 04 一月 2014
By 老逸

本系列博文记录学习Linux终端Bash过程中的收获，主要为常用命令一览表及相关功能的记录，方便日后再次查看。

常用过滤器

cat

就是把输入流或文件内容输出出来

cat [filename]

文件太长就可以用less分页显示

cat [filename] | less

tee

管道默认把输出作为下一级的输入， 某些情况下需要显示保存中间结果（管道很长之类的）， 可以使用tee

tee类似cat将输入作为标准输出，同时输出到文件

tac [filename] | tee [tempfilename] | tac | [......]

grep

Unix中的一个非常重要的过滤器。

在输入流中搜索并输出符合对应条件的行, 下面这货过滤出所有带'str'的行

cat [filename] | grep 'str'

设置过滤对大小写不敏感

grep -i [str]

反向过滤，找出所有不包含str的行

grep -v [str]

设置连带输出目标行后的4行（数字可变）

grep -A4 ......

设置连带输出目标行前的3行

grep -B3 ......

设置连带输出目标行前后的2行

grep -C2 ......

cut

截取并输出若干列的文本

设置分割符为':', 输出第一列和第三列的文本

cut -d: -f1,3 [filename]

设置分割符为空格时要用引号包括起来

cut -d" " -f1,2,3,5 [filename]

还可设置截取字符范围

截取每行第3到第7个字符

cut -c3-7 [filename]

tr

字符转换, 例如将小写a转为大写A

tr 'a' 'A'

设定范围转换

tr 'a-z' 'A-Z' #范围不一样会怎样呢？自己试试吧～

特殊字符也是可以的， 比如说把换行全部换成空格

tr '\n' ' '

还能用于压缩多个空格

tr -s ' '

移位什么的很方便

tr 'a-z' 'o-za-n'

删除所有a/b/c字符

tr -d 'abc'

wc

不是厕所，是字符计数的缩写

统计行数、字数及字符数

wc [filename]

sort

排序用的，默认是每行按字典序排 //突然想起当年瞎搞ACM的时候被字典序坑过好多次.-_-

sort [filename]

按数字大小排

sort -n [filename]

按某一列排(假设第三列)

sort -k3 [filename]

uniq

去掉相同的行

uniq [filename]

comm

逐行比较两个文件， 仅第一个有的在第一列， 仅第二个有的在第二列， 相同的在第三列

comm [file1] [file2]

可设置只显示某几列

comm -23 [file1] [file2]

od

可将文件内容编码成其他进制

转十六进制并输出,每词一个字节

od -t x1 [filename]

十进制输出

od -b [filename]

ASCII码输出

od -c [filename]

sed

使用正则表达式编辑文本。

基本的Unix工具

find

找东西用的，据说直接暴搜得到结果。 想要高效搜索用下面说的locate

查找某目录(及其子目录)下的所有文件

find [location]

查找某目录其子目录下的某种文件(匹配)

find [location] -name [option]

eg. find / -name "*.txt" #找系统中的所有txt文件

查找某种类型文件,关于type可查man

find [location] -type [type]

查找并对目标执行某些命令操作， -exec 后接操作的命令(下述例子删除了目标文件)

find ~ -name "*.txt~" -exec rm {} \;

locate

使用索引搜索文件， 比find效率更高

更新数据库(在我的Ubuntu里要sudo)

updatedb

查找文件

locate [pattern]

具体情况可以查man

date

看时间

date

自定义格式(A:星期 d:天 m:月 Y:年)

date +'%A %d-%m-%Y'

cal

月历

cal

显示某年某月的月历(1970年三月)

cal 3 1970

或者显示整年的日历

cal 1989

sleep

休眠嘛(单位：秒)

sleep 100 #别等它...

time

获取命令运行的时间

time echo 'hello,world'

gzip/gunzip:文件压缩工具

压缩目标文件， 默认得到同名的后缀为gz的文件。

注意：原文件会被替换

gzip [file]

将gz文件作为目标文件解压缩

gunzip [file] 或 gzip -d [file]

zcat命令可查看压缩包文本内容

zcat [file]
