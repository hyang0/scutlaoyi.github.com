Python笔记（三）

Date Wed 18 September 2013 By laoyi Category Python. Tags Python

本系列博文记录学习Python过程的笔记和想法，简单而狗血，杂乱而无章，主要目的是方便来日查看。

使用的Python版本为2.7.3， 在GNU/Linux Ubuntu 12.04LTS操作系统下实现和测试

函数

定义函数

关键词def，类似于C++，由函数名、参数列表、函数体和返回语句构成。

在def的下一行可添加字符串作为文档字符串。 文档字符串描述了函数的功能作用。 文档字符串作为函数的一部分存在。 可通过调用函数的__doc__属性返回文档字符串。

参数列表中的参数后可添加赋值语句，相当于C++中的默认参数。

调用函数时，默认通过位置复制，亦可指定参数名称赋值，称为关键字参数赋值。

基本例子如下：

def sayHello(a, b = 100):
        "This is a function to say hello!"
        print "a is", a, "b is", b
        return "hello"

print sayHello.__doc__
#This is a function to say hello!
sayHello(15)
#a is 15 b is 100
sayHello(100,50)
#a is 100 b is 50
print sayHello(b = 50, a = 60)
#a is 60 b is 50
#hello

与C++类似，基本类型（数字、字符串等）参数在函数内作为局部变量存在，改变其值或引用不影响外部实参。

传入列表实参

由于Python的鸭子类型，函数并不保证外部传入参数的类型，因此可传入列表，当传入的是列表或字典时，由于参数是列表或字典的引用，因此改变引用的列表的数据时，相当于改变了外部实参指向的列表，因此外部数据也会发生变化。

与上文函数不同，当不显式返回某个值时，函数会返回None值

def funcOne(listA):
        "This is a function to try the list parameter"
        listA[1] = 25
        return

print funcOne.__doc__
#This is a function to try the list parameter
listB = [1,2,3,4]
print listB
#[1, 2, 3, 4]
print funcOne(listB)
#None
print listB
#[1, 25, 3, 4]

参数收集

通过在参数前加入一个星号，参数自动转换为元组，接受外部调用时传入的所有剩余参数。具体见例子：

def funcTwo(arg1, *args):
        "This is a function using a tuple collecting parameters"
        print arg1,
        print args
        return

print funcTwo.__doc__
#This is a function using a tuple collecting parameters
funcTwo(1)
#1 ()
funcTwo(1,2,3)
#1 (2, 3)

上文例子中，调用funcTwo(1,2,3)，参数1传入arg1,剩余参数全部进入参数args，args本身作为元组出现。

类似的，在参数前加入两个星号可将参数变为字典，可接受任意数量的关键字参数

def funcThree(arg1, *arg2, **argMap):
        "This is a function using a tuple and a map to collect parameters"
        print arg1,
        print arg2,
        print argMap
        return

print funcThree.__doc__
#This is a function using a tuple and a map to collect parameters
funcThree(1)
#1 () {}
funcThree(1,2,3,4)
#1 (2, 3, 4) {}
funcThree(1, x = 2, y = 3, z = 4)
#1 () {'y': 3, 'x': 2, 'z': 4}
funcThree(1, 2, 3, 4, x = 5, y = 6, z = 7)
#1 (2, 3, 4) {'y': 6, 'x': 5, 'z': 7}

一目了然

反向操作

构建元组、列表或字典后，可通过在实参前添加一个或两个星号将单个实参拆分为多个参数传入函数中。

例如：

def funcFour(x, y):
        "a normal function which needs two parameters"
        print x, y

paraMap = {'x':1, 'y':2}
paraList = [3, 4]
print funcFour.__doc__
#a normal function which needs two parameters
funcFour(**paraMap)
#1 2
funcFour(*paraList)
#3 4

闭包

在Python中函数的作用域可以嵌套，在函数内可定义另一个函数，并将其作为返回值。

例如下文的例子，在外层函数中定义了一个内层函数，内层函数返回外层函数的参数arg1与内层函数的参数arg2的乘积，外层函数返回该函数。

简单分析之，里外层函数均拥有自己的数据作用域，外层函数调用结束后自身的作用域应该被释放。然而，调用外层函数获取一个包含了自身作用域的函数，其中拥有外层函数的部分变量数据，这种情况称为闭包。

其实自己还不是很懂，这货有待深究。

def funcCreator(arg1):
        "Function to create another function"
        def funcT(arg2):
                "Function created by creator"
                return arg1 * arg2
        return funcT

print funcCreator.__doc__
#Function to create another function
func1 = funcCreator(123)
print func1.__doc__
#Function created by creator
print "result of 123 * 5:", func1(5)
#result of 123 * 5: 615

scope

scope函数可获取当前状态下的数据空间，包含了变量和函数等基本信息。 要注意的是返回的字典不能随意更改，可能会造成未定义行为。

scope = vars()
print scope
