Bash笔记(一)

Date Thu 02 January 2014 By laoyi Category Linux. Tags Bash

本系列博文记录学习Shell终端Bash过程中的收获，主要为常用命令一览表及相关功能的记录。

基础知识

以echo为例，bash的基本命令及结构如下：

显示命令后的所有参数

echo [option] [arg0] [arg1] [arg2] [...]

命令、参数之间用空格区分，空格数量差异没有作用, 在单引号或双引号之间的所有字符被当成同一个参数

echo "Hey, this is a single argument."

设定option使echo解析字符操作符

echo -e "A line with n a newline."

命令执行成功返回0，否则返回非0值, 返回值的意义可查命令的man

命令的帮助文档

type

命令有两种类型，内部和外部命令 显示命令的类型

type [command]

某些命令有两种类型，显示所有类型

type -a [command]

当执行同名命令时内置命令优先， 执行外部命令必须指定命令路径，如echo:

/bin/echo Hello

man

查找某个命令的手册页

man [command]

查找某个配置文件的手册页

man [configuration file name]

模糊搜索:查找带string的命令列表

man -k [string]

匹配某个section的命令,关于section的解释在man man中有

man [section] [command]

查看当前系统的文件系统结构树

man hier

whatis

一句话描述命令

whatis [command]

whereis

查找命令的所在目录列表

whereis [command]

which

which查找命令的路径并确定命令所在目录

which [command]

目录操作

pwd

显示当前所在的工作目录

pwd

cd

进入某目录,可使用相对路径或绝对路径

cd [location]

进入home目录

cd ~

进入上一次的目录(方便！)

cd -

ls

显示当前文件夹内容

ls [location]

显示所有文件（包括隐藏文件）

ls -a [location]

显示详细信息

ls -l [location]

显示详细信息并令文件大小显示更人性化(-_-||)

ls -lh [location]

mkdir

创建目录

mkdir [dir_name]

一个命令创建多层目录

mkdir -p [dir_name/dir_name/......]

rmdir

当目录为空时删除该目录

rmdir [dir_name]

一个命令删除目录树

rmdir -p [dir_name/dir_name/......]

文件操作

file

显示文件类型

file [filename]

touch

创建一个新文件

touch [filename]

创建文件并指定时间

touch -t [time] [filename] //touch -t 201312291830 test

rm

删除文件(没了就没了啵...)

rm [filename]

删除前确认

rm -i [filename]

最牛逼的删除操作,强力递归删除目录及目录中的文件

rm -rf [dirname]

cp

复制文件,从原文件到目标文件

cp [source_filename] [destination_filename]

复制文件夹及其中的文件

cp -r [source_dir] [destination_dir]

同时复制多个文件到同一个文件夹(最后一个参数务必为一个文件夹)

cp [file1] [file2] [......] [destination_dir]

防止复制时的现有文件覆盖

cp -i [......]

复制时保留原文件的权限和时间信息

cp -p [......]

mv

移动文件（或者将文件改名）

mv [source_filename] [destination_filename]

文件内容操作

head

默认显示文件的前十行

head [filename]

可以设定显示的行数

head -[number] [filename] //head -4 test.txt

tail

默认显示文件最后十行

tail [filename]

同上，设置显示的行数

tail -[number] [filename]

cat

输出文件内容到标准输出中

cat [filename]

可同时输出多个文件

cat [file1] [file2] [file3] [...]

不指定文件，则从标准输入中读取数据，直到EOF, 可用这个特性创建新文件（利用输出重定向）

cat > out.txt

复制文件

cat in.txt > out.txt

tac

与cat相反的输出

less/more

翻页显示文件内容 //更推荐用less,可以使用类似vim的翻页操作，并且支持搜索

less [filename]

可将其他命令的输出管道输入到less中

ls | less

//关于管道后面有详细说明

strings

将某个二进制文件中的可读ASCII字符输出 //有点憋屈的命令...

strings /bin/ls

命令的执行与控制

alias

可为某个常用的命令设置别名，别名与原名等同

alias dog=cat

同样，可设定带option甚至arguments的别名，需要用引号包含

alias la='ls -a'

显示当前所有别名

alias

去除别名

unalias [command]

set -x

开启命令回显

set +x

关闭命令回显

semicolon

命令与命令之间用分号分隔

echo Hello; echo Hey

ampersand

单个与符号让当前命令在后台执行

sleep 20 &

dollar question

钱+问号获取前一个命令的返回值

echo Hey ; echo $?

//Hey 0

double ampersand

双与符号连接两个命令，第二个命令只在第一个命令执行成功时才执行

echo first && echo second

double vertical bar

双竖直分割符连接两个命令，第二个命令只在第一个命令失败(返回非0值)时执行

echo first || echo second

combination

组合使用&&和||可以做逻辑处理

rm file && echo ok || echo failed

pound sign

注释使用#号

escape character

使用反斜杠作为逃逸符

echo hello ; world

也可使用反斜杠作为隔行命令

echo hello world
