    用户管理

    Unix的用户列表在/etc/passwd中， 该文件为每个用户保存了一行信息， 包括用户名、用户id、组id、用户描述、用户的home目录及默认的shell信息

    用户密码加密后保存在/etc/shadow文件中， 包含用户名、用户密码、最小密码修改间隔时间、最大密码修改间隔时间、密码无效前提醒时间等。 时间的单位是天。

    用户的密码管理默认设置在/etc/login.defs中。 设置了包含上述几个时间在内的多个参数。

    识别用户

    whoami

    查看当前用户名

    who

    列表显示当前所有登录用户相关信息

    who am i

    这货是空格分开的，相当于who，只显示当前用户

    w

    这个能显示当前所有用户正在执行的工作

    id

    显示当前用户的用户id，组id等

    创建一个新用户scutlaoyi，设置home目录及用户描述

    useradd -m -d ...

    六 04 一月 2014

    本系列博文记录学习Linux终端Bash过程中的收获，主要为常用命令一览表及相关功能的记录，方便日后再次查看。

    常用过滤器

    cat

    就是把输入流或文件内容输出出来

    cat [filename]

    文件太长就可以用less分页显示

    cat [filename] | less

    tee

    管道默认把输出作为下一级的输入， 某些情况下需要显示保存中间结果（管道很长之类的）， 可以使用tee

    tee类似cat将输入作为标准输出，同时输出到文件

    tac [filename] | tee [tempfilename] | tac | [......]

    grep

    Unix中的一个非常重要的过滤器。

    在输入流中搜索并输出符合对应条件的行, 下面这货过滤出所有带'str'的行

    cat [filename] | grep 'str'

    设置过滤对大小写不敏感

    grep -i [str]

    反向过滤，找出所有不包含str的行

    grep -v [str]

    设置连带输出目标行后的4行（数字可变）

    grep -A4 ......

    设置连带输出目标行前的3行

    grep -B3 ...

    五 03 一月 2014

    本系列博文记录学习Linux终端Bash过程中的收获，主要为常用命令一览表及相关功能的记录，方便日后再次查看。

    变量

    dollar sign

    使用美元符标记变量, 变量大小写敏感。 常用的系统变量有：

    $HOSTNAME, $USER, $UID, $HOME

    $PS1

    变量定义终端的提示符, 可通过设置该变量改变提示符的内容：

    $PS1='\u@\h:\W$'

    显示内容：用户@主机:当前目录$

    $PATH

    设定系统搜索执行命令的路径

    使用自定义变量

    直接对变量名赋值即可

    MyVar=123

    使用时加上美元符

    echo $MyVar

    使用变量时引号的差异

    简单说来就是这样：

    echo '$MyVar' // $MyVar

    echo "$MyVar" // 123

    set

    查看当前所有变量

    set

    env

    查看当前exported的变量

    env

    unset

    取消设置某个变量

    unset ...

    基础知识

    以echo为例，bash的基本命令及结构如下：

    显示命令后的所有参数

    echo [option] [arg0] [arg1] [arg2] [...]

    命令、参数之间用空格区分，空格数量差异没有作用, 在单引号或双引号之间的所有字符被当成同一个参数

    echo "Hey, this is a single argument."

    设定option使echo解析字符操作符

    echo -e "A line with n a newline."

    命令执行成功返回0，否则返回非0值, 返回值的意义可查命令的man

    命令的帮助文档

    type

    命令有两种类型，内部和外部命令 显示命令的类型

    type [command]

    某些命令有两种类型，显示所有类型

    type -a [command]

    当执行同名命令时内置命令优先， 执行外部命令必须指定命令路径，如echo ...
