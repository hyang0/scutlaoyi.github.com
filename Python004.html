Python 笔记（四）

Date Mon 23 September 2013 By laoyi Category Python. Tags Python

本系列博文记录学习Python过程的笔记和想法，简单而狗血，杂乱而无章，主要目的是方便来日查看。

使用的Python版本为2.7.3， 在GNU/Linux Ubuntu 12.04LTS操作系统下实现和测试

类的初识

关于称呼的一些声明

本文以<Python基础教程>为基础，沿用了其中的部分称呼，一般的函数通称函数，类的成员函数称为方法。

但由于C++的一些思想比较根深蒂固，有时会混用两种称呼。

类的定义及创建对象

class SimpleClass:
        def funcA(self):
                print "Hey!Function A called!"
        def funcB(self):
                print "Hey!Function B called!"
        def funcC(self, name):
                self.name = name
                print "Name is set as %s" % name

sc1 = SimpleClass()
sc1.funcA()
#Hey!Function A called!
sc1.funcC("szy")
#Name is set as szy

一般方法

定义一般方法时需要将self作为第一个参数传入，在方法中调用self可以获得调用该方法的对象本身。

尝试过不添加self，发现可以运行，但通过对象或者通过类名均无法调用该方法。

调用一般方法可通过对象加方法名加参数列表，亦可通过调用类名加方法名并将对象作为第一个参数传入：

class SimpleClass:
        def funcA(self):
                print "FuncA"

sc1 = SimpleClass()
sc1.funcA()
#FuncA
SimpleClass.funcA(sc1)
        #FuncA

对象的特性（数据成员）

对象的数据成员（C++说法）直接通过赋值产生，如上文的self.name

每个对象拥有自己的作用域，因此一个对象的数据不影响其他对象

可在类定义的外部直接赋值新的数据成员

sc1.anotherArg = 'scutLaoYi'
print sc1.anotherArg
#scutLaoYi

Python没有真正的私有成员，所有成员均可在类外被调用，但可通过将成员名称改为 “__name__ ” 的风格隐藏该数据成员，通过这种方法类的外部无法直接调用 “__name__” 获取该数据成员。

类的静态数据成员

在类定义中不仅可以定义方法，还可直接在类作用域中声明变量

class SimpleClass:
        number = 14
        def funcA(self):
                print "Hey!Function A called!"

此处声明的number在类的作用域中，因此所有类成员共享一个number变量。调用类的静态数据成员可通过对象调用，亦可通过类名调用。

sc1 = SimpleClass()
sc2 = SimpleClass()
print sc1.number, SimpleClass.number
#14 14

通过在对象上赋值同名变量，可覆盖类的静态数据成员

sc2.number = 15
print sc1.number, sc2.number
#14 15
SimpleClass.number = 16
#修改的是静态成员的值
print sc1.number, sc2.number
#16 15

父类

声明父类的方法是在类名之后加入父类的名称，Python支持多继承，因此可以加入多个父类。子类继承父类的所有成员。

class SimpleClass:
        number = 14
        def funcA(self):
                print "FuncA"

class AnotherSimpleClass(SimpleClass):
        pass

asc1 = AnotherSimpleClass()
print asc1.number
#14
asc1.funcA()
#FuncA
